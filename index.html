Comparing Latitude's Relation

* This site shows the movement and changes in the weather as we move along various coordinates with respect to the equator.

* For starters, using some technical analysis in python, the latitudes and longtitudes are generated. Then using open source package (citipy), the city names and coordinates were located the cities closest to the 500 random coordinates.

* Then using API requests, the key variables like temperature, cloudiness, wind speed and humidity values were extracted (Courtesy: Weather API). Then we did a transfer of this data to our technical notebook and loaded the data by creating a dataset.

* With this data, scatter graphs of weather variables and city (based on longitude and latitude values) were analysed and observed. Explore the site to read more findings from the analysis.
